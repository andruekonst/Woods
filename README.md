## Installation

### Rust

1. Install `rustup`.
2. Set up nightly toolchain: ```
rustup toolchain install nightly
rustup default nightly
```

### Windows

1. Install boost with conda: `conda install -c conda-forge boost`
2. Run `python setup.py install --user` to build extension
